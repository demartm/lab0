# lab0
Постановка задачи:

1. Создать пустой проект в QtCreator, в процессе создания на одном из шагов, разместить проект под системой контроля версий git.

2. Открыть репозиторий через клиент GitKraken. В начальный момент времени, в клиенте, при открытии возникает запрос на
   инициализацию репозитория - выполнить данную команду.

3. Зафиксировать первые изменения в репозитории (Добавление файла проекта и файла исходника в репозиторий)

4. Создать ветку develop. Разработку ведем на ветке develop

5. Добавить предложенный код. Зафиксировать изменения.

6. Изучить код, понять какая задача решается. В начале кода, до подключения библиотек. Закоментированным текстом разместить описание решаемой задачи,
   т.е. сформулировать постановку задачи. Зафиксировать изменения.

7. Найти и исправить ошибки в коде. Зафиксировать изменения.

8. Протестировать код программы. Добавить тестовые случаи в виде хардкода. Зафиксировать изменения.

9. Обновить файл readme. Добавить постановку задачи. Зафиксировать изменения.

10. Слить в ветку master.
